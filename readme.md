## 打印当前的交易价与收益
	* 和功能界面一起打印，接着接收键盘输入
	* 交易价会变高或者变低，达到目标金额难度随天数增加
	* 新增项目后，生成的随机数数组应该增大
	* 收益可能性一开始较大，随后减少，最后趋于稳定
## 买进与卖出功能
	* 买进时存款减少当前交易价*交易量，卖出时存款增加当前交易价*交易量
	* 投资金额加减相同的量
## 播报功能
	* 存储开始时的存款与投资金额，在return 0；前播报
	* 达成成就时播报（仅首次达成）
## 随机事件
	固定词汇+随机数字形成的随机事件
	* 金额随机，不超出用户接受范围
	* 同时进行三个交易日（生成随机数的轮数由变量控制）
## 成就功能
	* 设置成就各自的flag，存入主函数定义的数组，在一个交易日选择结束后打印当前交易日达成的新成就（主函数用一个flag)，打印完置为0
	* 如果再遇到新成就，再置为1
## 新建游戏
	* 如果是新建游戏就执行初始化函数，
	* 如果不是就读取存档，存档用哈夫曼加密。
